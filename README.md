隐私政策

本隐私政策详细说明了我们在您使用“中国膳食指南”（以下简称“本应用”）时，关于收集、使用和披露您个人信息的政策和程序，并告知您的隐私权以及法律如何保护您。

我们使用您的个人数据来提供并改进服务。一旦使用本服务，即表示您同意按照本隐私政策收集和使用信息。

一、 定义与解释
账户：指为您创建的用于访问我们服务或部分服务的唯一账户。

本应用：指软件程序“中国膳食指南”。

公司：指“中国膳食指南”开发者（以下简称“我们”）。

个人数据：指与已识别或可识别的个人相关的任何信息。

使用数据：指在使用服务过程中自动收集的数据（例如页面访问时长）。

二、 收集和使用您的个人数据
1. 收集的数据类型
使用数据：在您使用服务时自动收集，包括设备IP地址、浏览器类型、访问页面、停留时间及唯一设备标识符。

位置信息：在获得您事先许可的情况下，我们可能会收集您的位置信息，用于提供个性化的膳食建议和功能。

健康与运动数据（特别说明）：为了通过华为运动健康接口为您生成精准菜单，在获得您明确授权后，我们将读取您的体重、步数和热量消耗情况。

2. 个人数据的使用目的
提供并维护服务：包括监控服务的使用情况。

管理账户：管理您作为服务注册用户的身份。

联系您：通过电子邮件、短信或推送通知，就功能更新、安全提醒等必要信息与您联系。

膳食推荐：根据您的身体数据（如体重、消耗）结合《中国居民膳食指南》生成每日三餐建议。

三、 数据的存储与删除
保留时间：我们仅在实现本隐私政策所述目的所必需的期间内保留您的个人数据。

删除权利：您有权删除或要求我们协助删除我们收集的关于您的个人数据。您可以通过应用内的设置菜单或直接联系我们来操作。

安全性：我们采用商业上合理的手段保护您的数据，但请注意，互联网传输或电子存储并非100%安全。

四、 未成年人隐私
本服务不面向 13 岁以下的任何人。我们不会故意收集 13 岁以下未成年人的个人信息。如果我们发现意外收集了未成年人的数据且未经监护人同意，我们会立即从服务器中删除。

五、 第三方链接与服务
本应用可能包含指向第三方网站的链接（如美团买菜相关内容）。我们对第三方网站的内容、隐私政策或惯例不承担任何责任，建议您阅读其各自的隐私政策。

六、 本隐私政策的变更
我们可能会不时更新隐私政策。如有变更，我们会在本页面发布新的隐私政策，并更新页首的“最后更新日期”。建议您定期查看。
